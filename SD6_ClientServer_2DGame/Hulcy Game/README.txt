Starting the Server:
	Open the server.bat and change the ip address to the ip address the server will be running on.
	Run the server.bat file first to start the server

Starting a client:
	Open the client.bat and change the address to the address the server is running on.
	Then run the client.bat file.
	If the client fails to connect to the server open the console by pressing '~'
	Then use the console command "connect <ip address>" to retry the connection
